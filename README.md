## 📁 About this project

This repository provides software and scripts to set up Linux Mint version 22 and newer releases. It helps users configure their desktop environment after a fresh installation. The tools included here automate common tasks, install helpful applications, and adjust system settings to improve your experience. 

You do not need programming knowledge to use these files. This project focuses on stability and performance for your daily desktop tasks.

## 💻 System requirements

Before you begin, ensure your computer meets these requirements:

* An installed copy of Linux Mint 22 or a newer version based on Ubuntu 24.04.
* An active internet connection to download extra packages.
* At least 2 gigabytes of free disk space.
* You need administrator rights for your computer account.

## ⚙️ How to install and run

Follow these steps to set up the configuration tools:

1. Open your file manager. 
2. Right-click the file you downloaded.
3. Select "Extract Here" from the menu. This creates a new folder.
4. Open the new folder.
5. Look for the file named `install.sh` or `setup`.
6. Right-click this file and choose "Properties."
7. Go to the "Permissions" tab.
8. Check the box that says "Allow executing file as program."
9. Close the window.
10. Right-click the file again and select "Run." 

A terminal window opens to show the progress. You might see a prompt asking for your password. Type your login password and press Enter. The characters do not appear on the screen for security reasons.

## ❓ Frequently asked questions

**Does this change my personal files?**
No. The scripts only affect system-level folders and settings. Your personal documents, photos, and music remain in your home folder and stay untouched.

**What if I encounter an error?**
The terminal window displays information if a step fails. Copy the text from the window and check the issue tracker on this repository. Most errors relate to missing internet access or a lack of administrative permission.

**Can I undo these changes?**
Most settings allow you to revert to the system defaults through the Linux Mint "System Settings" menu. The cleanup tool only deletes temporary files, so you cannot undo that process, but you will not lose data.

**How often should I run this?**
You only need to run the installer once after your initial operating system installation. You can run the update and cleanup specific scripts whenever your system feels slow or before a major system upgrade.
